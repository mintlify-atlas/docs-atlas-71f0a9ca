## Quick Start

Get up and running with Synkra AIOX in minutes.

<Steps>
  <Step title="Install the Framework">
    <CodeGroup>
    ```bash New Project
    npx aiox-core init my-project
    cd my-project
    ```
    ```bash Existing Project
    cd your-project
    npx aiox-core install
    ```
    </CodeGroup>
  </Step>
  
  <Step title="Activate an Agent">
    Choose your IDE and activate an agent:
    - **Claude Code**: `/dev` or `/qa`
    - **Gemini CLI**: `/aiox-menu` → select agent
    - **Codex CLI**: `/skills` → `aiox-dev`
  </Step>
  
  <Step title="Start Building">
    Run agent commands to begin your workflow:
    ```bash
    *help           # View available commands
    *create         # Create new stories
    *develop        # Start development
    ```
  </Step>
</Steps>

## Why Synkra AIOX?

<Note>
**Agentic Planning + Engineering-Contextualized Development**

Synkra AIOX eliminates the two biggest problems in AI-assisted development: planning inconsistency and context loss. Specialized planning agents create detailed specifications, then the Scrum Master transforms them into hyper-detailed stories containing everything the dev agent needs.
</Note>
